# 设计规范

## 设计理念
- **移动优先**：所有设计以手机屏幕为基准
- **触控友好**：每行至少 48px 高度，确保手指可精确点击
- **深色科技风**：与简历网站风格统一，暗色背景 + 霓虹点缀
- **简约直观**：减少学习成本，打开即用

## 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景色（深）| `#0A0E1A` | 主背景 |
| 卡片背景 | `rgba(15,20,40,0.75)` | 毛玻璃效果 |
| 主文字 | `#E8ECF4` | 高可读性 |
| 次要文字 | `#9CA3B8` | 辅助信息 |
| 强调色（青）| `#00E5FF` | 主要交互、翻译结果 |
| 强调色（绿）| `#69F0AE` | 成功状态、原文行 |
| 边框 | `rgba(100,140,255,0.25)` | 卡片/输入框边框 |

## 排版规范

| 元素 | 字号 | 字重 | 说明 |
|------|------|------|------|
| 页面标题 | 22px | 700 | 顶部标题 |
| 模式标签 | 14px | 600 | 翻译方向/模式 |
| 原文行 | 16px | 400 | 可点击的文本行 |
| 翻译结果 | 16px | 500 | 翻译后显示 |
| 按钮文字 | 15px | 600 | 操作按钮 |
| 辅助文字 | 12px | 400 | 提示/说明 |

## 间距规范

| 场景 | 间距 | 说明 |
|------|------|------|
| 行与行之间 | 12px | 点击区域间隔 |
| 行内边距 | 16px 14px | 上下16px 左右14px |
| 卡片内边距 | 20px | 内容区 |
| 卡片间距 | 16px | 卡片之间 |
| 输入框高度 | 120px | 粘贴区域 |

## 触控交互

### 文本行状态
```
[默认] 暗色背景 + 白色文字 + 左侧指示条
  ↓ 点击
[加载] 骨架屏闪烁 0.3s
  ↓ API返回
[翻译] 青色高亮边框 + 翻译文字淡入
  ↓ 再次点击
[恢复] 回到原文状态
```

### 动效规范
- 翻译淡入：`opacity 0→1, 0.3s ease`
- 行点击反馈：`background 颜色闪烁, 0.15s`
- 模式切换：`transform slide, 0.3s ease`
