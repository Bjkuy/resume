# 开发需求文档

## 项目名称
手机翻译小工具（Mobile Translator）

## 项目概述
一个运行在手机上的翻译工具，用户可以粘贴文字或输入网址，文字自动分行显示，**点击任意一行**即可翻译成目标语言。

## 目标用户
- 不懂代码的普通用户
- 需要快速翻译英文内容的手机用户
- 浏览英文网页需要辅助理解的人

## 功能需求

### F1：粘贴翻译模式
| 编号 | 描述 | 优先级 |
|------|------|--------|
| F1.1 | 用户可将文字粘贴到输入框 | P0 |
| F1.2 | 文字按段落/句子智能分行展示 | P0 |
| F1.3 | 点击任意一行，该行立即翻译为目标语言 | P0 |
| F1.4 | 再次点击已翻译的行，可切换回原文 | P1 |
| F1.5 | 支持一键翻译全部行 | P1 |

### F2：网址翻译模式
| 编号 | 描述 | 优先级 |
|------|------|--------|
| F2.1 | 用户可输入/粘贴网址 | P0 |
| F2.2 | 自动抓取网页文字内容 | P0 |
| F2.3 | 提取正文（去除导航、广告等干扰）| P1 |
| F2.4 | 分行显示，支持点击翻译 | P0 |

### F3：翻译方向切换
| 编号 | 描述 | 优先级 |
|------|------|--------|
| F3.1 | 支持英文 → 中文翻译 | P0 |
| F3.2 | 支持中文 → 英文翻译 | P0 |
| F3.3 | 自动检测输入语言 | P2 |
| F3.4 | 一键切换翻译方向 | P0 |

### F4：移动端体验
| 编号 | 描述 | 优先级 |
|------|------|--------|
| F4.1 | 手机浏览器完美适配 | P0 |
| F4.2 | 行间距足够大，手指可精确点击 | P0 |
| F4.3 | 可添加到手机桌面（PWA）| P1 |
| F4.4 | 点击有视觉反馈（动效）| P1 |

### F5：分享与部署
| 编号 | 描述 | 优先级 |
|------|------|--------|
| F5.1 | 通过 GitHub Pages 托管 | P0 |
| F5.2 | 链接可在微信中打开和分享 | P1 |
| F5.3 | 好友可直接使用，无需安装 | P0 |

## 非功能需求
- 翻译 API 免费使用，无需用户注册/付费
- 页面加载快（3秒内可用）
- 移动端优先，触控友好
- 支持弱网环境（PWA 缓存）
